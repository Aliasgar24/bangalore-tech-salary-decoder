# Bangalore Tech Salary Decoder

## Overview

This project analyzes salary data of Bengaluru tech professionals using Python and Pandas. The dataset is cleaned, standardized, and analyzed to answer business questions related to salaries, experience, skills, company types, and compensation trends.

## Features

* Data cleaning and preprocessing
* Standardization of role names and company types
* Salary analysis by job role
* Experience-wise salary growth analysis
* Skill premium analysis
* Company type salary comparison
* Identification of underpaid professionals
* Final business report with insights

## Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## Dataset

The dataset contains salary information for approximately **1,000 Bengaluru tech professionals** with details such as:

* Job Role
* Years of Experience
* Current CTC
* Previous CTC
* Company Type
* Skills
* Education Tier
* Work Mode

## Project Workflow

1. Load and inspect the dataset
2. Clean and standardize the data
3. Perform business analysis
4. Generate insights
5. Create the final report

## Key Insights

* Senior Product Managers had the highest median salary.
* Backend salaries increased significantly with experience.
* System Design showed the highest salary premium among the selected skills.
* Unicorn companies offered a higher median salary than MNCs for SDE Backend professionals.
